# API Monitor

Real-time API monitoring dashboard built with Angular 19 using standalone APIs, Signals, and a scalable feature-based architecture.

---

## Features

- API endpoint monitoring
- Response time tracking
- Dynamic health status updates
- Endpoint management interface
- Reactive state management with Signals
- Modern standalone Angular architecture
- Responsive dashboard UI with Tailwind CSS

---

## Tech Stack

### Frontend
- Angular 19
- TypeScript
- Tailwind CSS
- Angular Signals

### Backend 
- NestJS
- PostgreSQL

---

## Project Structure

```bash
src/app
├── core
│   └── services
├── shared
├── features
│   ├── dashboard
│   └── endpoints
```

---

## Getting Started

### Install dependencies

```bash
npm install
```

### Start development server

```bash
ng serve
```

Application runs at:

```bash
http://localhost:4200
```

---

## Build

```bash
ng build
```
